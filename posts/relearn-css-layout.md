Understanding CSS layout has always been hard. Nowadays with CSS grid and flexbox it's perhaps even harder. Even for the folks who have been 
using floats for most of their development life. Andy Bell and Heydon Pickering created this awesome website where they (re)learn
you everything about CSS layout. Go check it at [every-layout.dev](https://every-layout.dev/).
