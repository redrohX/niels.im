It took some effort, but I have [Indiekit](https://getindiekit.com/) in some basic form running on a VPS. Which makes it possible to publish posts with Micropub on my personal website and (optionally) syndicate these posts to Mastodon. This is my first real post I’m writing with the Indiekit interface. Which is basically a CMS, but because of Micropub, makes it also possible to use different Micropub clients. I haven’t tried this yet.
